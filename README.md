# Electro.pk - Premier Electric Store in Pakistan 🏪

**Electro.pk** is a modern, SEO-optimized React e-commerce website for an electrical supplies store based in Pakistan. The website allows customers to browse and order electrical products including fans, lights, switches, wires, and more.

## ✨ Features

### Core Functionality
- 🛒 **Full Shopping Cart** - Add products, update quantities, remove items
- ❤️ **Wishlist** - Save products for later
- 🔍 **Product Search** - Search across all products and categories
- 📱 **WhatsApp Orders** - Direct WhatsApp integration for quick orders
- 💳 **Multiple Payment Options** - Cash on Delivery, JazzCash, EasyPaisa, Bank Transfer

### Pakistan-Specific Features
- 🇵🇰 **PKR Currency** - All prices in Pakistani Rupees (Rs.)
- 🚚 **Nationwide Delivery** - Delivery to all major Pakistani cities
- 📱 **WhatsApp Integration** - Direct ordering via WhatsApp
- 🕐 **Local Business Hours** - 9 AM - 9 PM, 7 days a week

### SEO Optimization
- 📝 **Meta Tags** - Comprehensive title, description, keywords
- 🔗 **Open Graph Tags** - Social media optimization for Facebook, Twitter
- 📊 **Structured Data (JSON-LD)** - Rich snippets for search engines
- 🗺️ **Sitemap** - XML sitemap for search engines
- 🤖 **Robots.txt** - Proper crawling instructions
- 🏢 **Local Business Schema** - Google business listing

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ and npm

### Installation

```bash
# Clone or navigate to the project
cd electro-pk

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 📦 Available Scripts

| Command | Description |
|---------|-------------|
